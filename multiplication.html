Multiplication !

Lesson !!! !

Multiplication, in its simplest form is repeated addition. For example, instead of saying 2+2+2, you would say 2x3, this is because 2 is being added 3 times. This is the same as 3+3, instead, you could write 3x2 because 3 is being added twice, and again for 1, 1+1+1 will become 1x3, because you are adding 1, 3 times. This is true until you multiply numbers by 0. This is not the case for 0 though, any number multiplied by 0 is automatically 0, 0x1 is 0, 0x20 is 0, and 0x1000 is 0, no matter what, if 0 is in a multiplacation equation, the answer is always 0. In multiplication, the answer to the problem isn't a sum, but rather the product, and the numbers in the questions are factors. So in 2x2, 2 and 2 are the factors and 4 (being the answer) is the product. There are many ways to remember Multiplication but the easiest is by practicing the times table and by practicing what every indivisual numbers products are. You study 2x2 then 2x3, then 2x4 all the way to 10. When you have memorized that you can then move onto 3.
